### Использованные технологии

- Семантическая верстка HTML5
- CSS Custom Properties (Переменные)
- Flexbox
- CSS Grid
- Адаптивные изображения (тег `<picture>`)
- JavaScript (для мобильного меню)

### Чему я научился

В этом проекте я закрепил навыки работы с CSS Grid и Flexbox. Особенно интересным было создание мобильного меню с треугольником на чистом CSS и реализация адаптивности через `grid-template-columns`.
